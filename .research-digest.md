# Daily Research Digest - 2026-04-07

## CVEs and Security Advisories

### HIGH PRIORITY

**BIND DNS (ns1) — Multiple CVEs, patched in 9.18.48 / 9.20.22**
- CVE-2026-3591 (High): Stack use-after-return in SIG(0) handling — ACL bypass, potential IP mis-match
- CVE-2026-3119: named crash when processing TKEY record in correctly-signed query
- CVE-2026-1519 (CVSS 7.5): Targets BIND resolvers performing DNSSEC validation
- CVE-2026-3104: Memory leak in recursive resolvers
- Action: Apply Rocky Linux BIND errata on ns1 (targets 9.18.48 LTS)

**Authentik — CVE-2026-25227 (Critical RCE) + CVE-2026-25748 (Auth Bypass)**
- CVE-2026-25227: Users with "Can view Property Mapping/Expression Policy" can execute arbitrary code via test endpoint — full DB access. Fixed in 2025.8.6 / 2025.10.4 / 2025.12.4
- CVE-2026-25748: Forward auth bypass with Traefik/Caddy reverse proxies. Fixed in 2025.10.4+
- Action: Verify running Authentik version; upgrade to at least 2025.10.4

**n8n — CVE-2026-21858 (CVSS 10.0 RCE) + CVE-2026-25049 (Authenticated RCE)**
- CVE-2026-21858: Content-Type confusion in webhook/file-handling allows unauthenticated RCE. Fixed in v1.121.0
- CVE-2026-25049: Authenticated users can abuse expressions for system command execution. Fixed in v1.123.17 / v2.5.2
- Action: Verify n8n version on server01 is >= 1.123.17
