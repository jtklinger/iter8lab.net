# Daily Research Digest - 2026-04-02

## CVEs and Security Advisories

### URGENT

**Wazuh — CVE-2026-25790 (High, Stack Buffer Overflow / DoS/RCE)**
- Affects versions 3.9.0 through **4.14.3** — current deployed version IS VULNERABLE
- Stack-based overflow via `sprintf` with floating-point specifier on fixed-size buffer in SCA JSON parser
- A crafted JSON event can crash or potentially RCE the Wazuh manager
- Fixed in: **4.14.4** (released March 17, 2026)
- Action: Upgrade wazuh-manager on kvm02 to 4.14.4 immediately

**BIND — 4 CVEs disclosed March 25, 2026 (2x High, 2x Medium)**
- CVE-2026-3104 (High): Memory leak in DNSSEC non-existence proofs — crafted domains cause unbounded RSS growth / OOM crash
- CVE-2026-1519 (High): Excessive NSEC3 iterations during insecure delegation validation cause high CPU / DoS
- CVE-2026-3591 (Medium): Stack use-after-return in SIG(0) handling — ACL bypass / IP mis-matching
- CVE-2026-3119 (Medium): named crashes on valid TSIG-signed query with TKEY record
- Fixed in: 9.18.47, 9.20.21
- Action: Check BIND version on ns1, patch if below 9.18.47

### HIGH

**n8n — CVE-2026-33663 (High, Credential Theft)**
- `global:member` role users can steal plaintext secrets from other users' credentials
- Affects Community Edition; fixed in 1.123.27+, 2.13.3+, 2.14.1+
- Also: CVE-2026-21858 (Critical RCE, fixed in 1.121.0+) and CVE-2026-25049 (Critical RCE, fixed in 1.123.17+)
- Action: Verify n8n version on kvm02, upgrade if below 1.123.27

**Netbird — 2 Authorization Vulnerabilities (now patched in latest)**
- AIKIDO-2026-10229: Account impersonation via `?account=` parameter bypass in management API — fixed in 0.64.6+
- AIKIDO-2026-10286: Race condition allows privilege escalation — fixed in 0.65.3+
- Latest release: v0.67.2 (April 1, 2026) — health check flags, auto-restart on IP change, DNS/IPv6 fixes
- Action: Verify netbird-server is on v0.65.3+ (ideally v0.67.2)
- Note: Active spear-phishing campaign abusing NetBird for persistent VPN access reported by Centripetal

**Authentik — CVE-2026-25227 (RCE) + CVE-2026-25748 (Auth Bypass)**
- CVE-2026-25227: Any user with "Can view Property Mapping" permission can execute arbitrary code inside authentik container (full takeover)
- CVE-2026-25748: Traefik/Caddy forward auth users can craft malformed cookie to bypass auth entirely
- Fixed in: 2025.10.4, 2025.12.4
- Note: Authentik not yet deployed in homelab/OurHomePort — relevant for future deployment planning
